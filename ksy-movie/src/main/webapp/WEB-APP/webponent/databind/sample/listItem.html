databind="_this.listItem(array)"
TARGET : 바인더 하위의 TBODY, UL, OL Element
  1. <tbody data-bind="_this.listItem(a.block)"></tbody>
  2. <ul data-bind="_this.listItem(a.block)"></ul>
  3. <ol data-bind="_this.listItem(a.block.reverse())"></ol>

LIST성격의 element를 처리 하기 위한 미리 준비된 함수로 array data를 인자로 받는다. 실시간 표현식의 경우도 _this.appendItem(array) 또는 _this.replaceItem(array,유니크키)등이 준비되어 있다.
