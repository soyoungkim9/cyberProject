data-max-item="2"
TARGET : 바인더 하위의 TBODY, UL, OL Element
  1. <tbody data-bind="_this.listItem(a.block)" "></tbody>
  2. <tbody data-bind="_this.listItem(a.block)" data-max-item="2""></tbody>
  3. <tbody data-bind="_this.listItem(a.block)" data-max-item="3^10""></tbody>

listItem을 처리중에 data-max-item를 설정하면 최대 출력 갯수를 설정할 수 있다. 없는경우는 전부 출력한다.

  • data-max-item="2" ==> 조회시는 2개만 출력, 실시간은 BINDER_CONFIG.LIMIT_APPENDITEM_SIZE로 설정
  • data-max-item="3^10" ==> 조회시는 3개만 출력, 실시간은 10개가넘어가면 삭제하여 10개로 맞춤
  • data-max-item="^10" ==> 조회시는 전부 출력, 실시간은 10개가넘어가면 삭제하여 10개로 맞춤
